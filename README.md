# 测试平台Demo样例（建设中）

## 简介

用来做数据库设计和部分功能调试的 Demo

当前测试平台样例的后台框架选用 django

## 安装

- [ ] 安装 python 3.12 版本
- [ ] 使用 pip 安装三方库
- [ ] 修改 testsite/settings.py 中的测试数据库（因为当前数据库部署在本机，待迁移）

```
pip install -r requirements.txt
```
